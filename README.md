# Client for new Tilos Radio website

The application based on [Sapper](https://github.com/sveltejs/sapper).

## First run

```bash
nvm use
npm i
```

## Develop

```bash
npm run dev
```

You can access the application on the following URL [localhost:3000](http://localhost:3000)

## Test
```bash
npm run test
```
or
```bash
npm run dev
npm run cy:open
```
